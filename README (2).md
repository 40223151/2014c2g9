# 課程合併

自 2014 年 Fall 起, 程式語言(一)與(二)將合併為計算機程式課程, 
大一下接續的選修課則為網際網路應用與網際內容管理, 大二上必修課程為
電腦輔助設計實習, 大二下則為協同產品設計實習, 大三上選修課為產品資料管理, 
大三下則為產品生命週期管理.
